# Игра «Змейка»

Классическая игра «Змейка» на Pygame.

## Установка зависимостей

Если pygame не установлен, выполните:

```bash
pip install -r requirements.txt
```

или

```bash
pip install pygame
```

## Запуск игры

```bash
python3 snake_game.py
```

или

```bash
python snake_game.py
```

## Управление

- **Стрелка вверх** ↑ - движение вверх
- **Стрелка вниз** ↓ - движение вниз
- **Стрелка влево** ← - движение влево
- **Стрелка вправо** → - движение вправо

## Правила игры

- Змейка движется автоматически в выбранном направлении
- Собирайте красные яблоки, чтобы увеличить длину змейки
- При столкновении с собой игра начинается заново
- Змейка проходит сквозь стены и появляется с противоположной стороны

## Выход из игры

Закройте окно игры или нажмите Ctrl+C в терминале.
